Modules:

projects_manager.py (Gather the data for new ini files, forwards it for creating a new one or overwriting.)
config_manager.py (Loads ini files and/or overwrites them with ProjectsManager()'s help. Also sets up local user data folders for migration and validation.)
version_logic.py (Creates/updates the changelogs of the projects)
opener.py (Cross-platform file and folder opening utility)

==Startup routine==
1. Identify the current execution context
2. Verify the existance and integrity of local user data
3. Initialize the environment if missing or unsafe
4. Inspect and resolve .tmp files from last sessions
5. Set up shared state and enter the main loop
